Frequently asked questions

Built-in details/summary means no JS needed for accordions.

› Can I use this template commercially?

Yes. It’s a starter you can adapt and ship for client or personal projects.

› How do I change the colors?

Edit CSS variables in :root (primary, secondary, accent) to theme quickly. Light/dark adapts automatically via prefers-color-scheme.

› Will it work without JavaScript?

Absolutely. All animations and interactions here are HTML/CSS only, including the mobile navigation.
